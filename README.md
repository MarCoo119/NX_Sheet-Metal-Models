## Sheet Metal CAD Models – Siemens NX
## 📌 Overview

This repository contains sheet metal CAD models created using Siemens NX, focusing on standard industrial design and manufacturing practices. The models demonstrate correct use of NX sheet metal features and workflows.

## 🛠️ Features

Designed using Siemens NX Sheet Metal module

Proper sheet thickness and material consistency

Standard features: bends, flanges, hems, cutouts, and reliefs

Accurate bend allowance and flat pattern generation

Fully parametric and editable models

## 📂 Repository Contents

Individual sheet metal part files

Folded models and corresponding flat patterns

Manufacturing-ready geometry for laser cutting and bending

## 🎯 Purpose

Learning and practicing Siemens NX sheet metal design

Academic projects and training

Portfolio and demonstration of CAD skills

## 🧰 Tools & Software

Siemens NX (Sheet Metal Design)

Parametric sketching and feature-based modeling
